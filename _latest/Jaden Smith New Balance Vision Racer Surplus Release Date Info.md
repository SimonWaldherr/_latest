<p>In collaboration with brand ambassador Jaden Smith, New Balance is excited to launch the Vision Racer &#8216;Surplus&#8217; exclusively with&#160;Greenhouse.</p>
<p>Jaden Smith&#8217;s <a href="https://www.sneakerfiles.com/sneakers/new-balance-vision-racer/">New Balance Vision Racer</a> merges classic NB design with Jaden&#8217;s unique, bold, fashion-forward aesthetic for a silhouette that pushes the boundaries for sustainably-led footwear at New Balance. The NB for Jaden Smith Vision Racer &#8220;Surplus&#8221; leverages surplus textiles and synthetic leathers, offering consumers a bespoke silhouette that takes Jaden&#8217;s vision to have as little impact as possible with materials to the next level.</p>
<p>The use of surplus materials was driven by Jaden Smith&#8217;s vision to have as little impact as possible with materials. In the first release of the NB for Jaden Smith Vision Racer, New Balance explored this idea using recycled content. With the &#8220;Surplus&#8221; style, the concept goes to the next level by using surplus textiles and synthetic leathers for the upper of the shoe to create a bespoke, limited model &#8211; with no two shoes, let alone pairs, the same. The NB for Jaden Smith Vision Racer &#8220;Surplus&#8221; continues to be vegan-friendly even with the randomized materials.</p>
<h2>Jaden Smith New Balance Vision Racer Surplus Release Details</h2>
<p>Expect Jaden Smith&#8217;s <strong>New Balance Vision Racer &#8216;Surplus&#8217;</strong> to release on October 21st exclusively on the Greenhouse app. A wider release will take place on October 23rd via <a href="https://www.footlocker.com/" class="external" rel="nofollow" target="_blank">FootLocker.com</a> and select European retailers. The retail price is $150. More images below that will give you a better look.</p>
<p><strong>New Balance Vision Racer Surplus</strong><br>
Release Date: October 23, 2020<br>
Price: $150</p>
