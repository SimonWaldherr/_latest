<div><ul class="summary-list">
<li>You can search a conversation on Skype to find particular words or phrases in text chats with contacts.&#160;</li>
<li>Skype's search or "find" feature will only review text from the last six months.&#160;</li>
<li>The search function is available on both the desktop and mobile versions of Skype.</li>

<p>If you've ever had a conversation with someone via Skype and later wanted to recall a nugget of what was said, Skype's Search feature helps you find what you're looking for.&#160;</p><p>You can search through any Skype conversation on either the mobile or desktop applications. Here's how to do it.</p><h2>How to search a conversation on Skype using its desktop application</h2><p><strong>1.</strong> Click on the Skype application and sign in.&#160;</p><p><strong>2.</strong> In the Skype application window, select the conversation you wish to search from the left column of the desktop app window.&#160;</p><p>
